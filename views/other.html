Other Threats

Some other threats that you should be aware of.

Examples of Other Threats

Weak Passwords

Weak passwords are a hacker's easy way into your computer or accounts. You can see the strength of your password at security.org.

Misconfigured Access Rights

Computers have access rights which give different users different permissions on the computer system. Get it wrong and a guest could possibly install malicious software.

Removable Media

Like USB Sticks and Portable Hard Drives. They can transfer malicious software.

Unpatched Software

Software Developers and Companies release patches for any security issues. If you hasn't installed the patch, your computer has a vunerability.
