Malicious Software and Code

Examples of Malicious Software and Code

Viruses

Software that is loaded on to your computer without your knowledge or proper authorisation. It performs mailicious activity such as saving files, key logging, self-replicating, etc.

Trojans

Based of the story from Greece of the Trojan Horse, where in the Trojan War, the Greek soldiers entered Troy in a structure shaped like a horse which the residents of Troy let in, which led to Greece winning the war. Software that looks real and legitimate, but really does malicious activity in the background.

Adware

Software that shows unwanted popus and adverts.

Spyware

Installed software that spys on the user of the computer (e.g. keylogging).
