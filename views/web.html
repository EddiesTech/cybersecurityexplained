Web Specific Threats

Being a webmaster is hard... And then there are all these threats you need to keep on top of to keep your website alive and safe.

Image by Christiaan Colen (CC BY-SA 2.0)

Examples of Web Specific Threats

Cross Site Scripting (XSS)

When user input is rendered as unescaped HTML, meaning users can run potentially harmful code on your site, without your authorisation.

SQL/NOSQL Injections

When user input is stored in a database but they can inject code that can do harmful things to the database if not seperated or validated.
